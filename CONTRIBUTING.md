# Contributing to TPT IT COM

Thank you for your interest in contributing to **TPT IT COM**! We welcome developers, designers, and testers of all skill levels to help improve this open-source project.

---

## Code of Conduct
By participating in this project, you agree to maintain a respectful, inclusive, and professional environment. Please refrain from using offensive language, harassing others, or engaging in discriminatory behavior. We look forward to creating a collaborative space for everyone.

---

## Contribution Guidelines
To ensure consistency and quality across the codebase, please keep the following guidelines in mind:
* Maintain clean, readable code and use intuitive variable names.
* Comment complex logic within your components where necessary.
* Ensure all styles remain modular and responsive across desktop and mobile screens.
* Run tests locally before submitting any changes to verify everything works seamlessly.

---
